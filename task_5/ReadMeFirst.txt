This folder contains the following documents & subfolders:*Raw Folder	This folder consists of the raw data in .csv and .dta format. There are three datasets: "growthdata92_02", "growthdata02_11" and “pen_jae”. *Analysis Folder	This folder contains Code and Output subfolders:		-Code contains the do files for the Exercise 1(cross_fit_po.do) and a subfolder named Exercise 2 where you can find the do files for the three sub-exercises.		-Output consists of data files for each sub-exercise of Exercise 2 that show the MSE of different prediction methods and also a latex file(cross_fit_po.tex) for the table generated for Exercise 1. *Communications Folder	It contains a final pdf document for the task 5(Task_5.pdf) where you can find the comments for each exercise, and also a Log file which has the log files for each exercise.
